# KokoScript  
*Speak the Bible with local TTS — no cloud required.*

A self-hosted, voice-enabled Bible reading assistant powered by free APIs and TTS models.  
Designed to run entirely offline on your own machine — perfect for privacy-focused users, low-bandwidth environments, or quiet devotional time.

## Features
- Reads Bible verses from APIs (Should require limited data)
- Uses local TTS models (e.g., Coqui TTS) for natural voice generation
- Easy to deploy with Docker Compose
- Outputs audio in real-time

## Requirements
- Docker and Docker Compose
- An internet connection?

## Setup

1. Clone this repo:
   ```bash
   git clone https://github.com/taltoris/KokoScript
   cd KokoScript
2. docker compose up --build
3. Point your browser to localhost:5003
4. Install Kokoro, Use Now, Stream!
